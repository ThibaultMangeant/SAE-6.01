# CENOFM
## Contexte : SAE 6.01 Evolution d'une application existante

### Pré-requis

- Java SDK 24 au minimum

### Lancement de l'application

- Sur Windows :<br>
Executez le fichier `setup.bat` soit :
  - en double cliquant dessus dans un explorateur de fichier
  - en executant la commande `./setup.bat` dans un terminal de commande.

- Sur Linux :<br>
Executez le fichier `setup.sh` (exclusivement avec un terminal de commande) :
  - Executez la commande `chmod u+x chemin/vers/setup.sh` si c'est la première fois.
  - Executer le fichier avec `./setup.sh`.
